# 🎨 نظام اختيار نوع توليد الصور

## تحديث جديد: اختيار نوع التوليد من قبل المستخدم

تم إضافة **خيار اختيار نوع توليد الصور** مباشرة من واجهة المستخدم. المستخدم يمكنه الآن أن يختار بين ثلاثة خيارات:

---

## ✅ الخيارات الثلاثة:

### 1. 🔄 **تلقائي (Auto)** - الخيار الافتراضي
- **الوصف:** يحاول توليد صور حقيقية أولاً عبر الإنترنت
- **السلوك:** 
  1. يحاول Pollinations (AI generation)
  2. يحاول Pixabay (real photos search)
  3. يحاول Unsplash (real photos search)
  4. إذا فشلت جميع المحاولات → يستخدم **التوليد المحلي كخيار أخير**
- **الفائدة:** أفضل جودة ممكنة مع ضمان عدم الفشل الكامل
- **متطلبات:** اتصال إنترنت (لكن يعمل بدونه كبديل)

### 2. ☁️ **حقيقي فقط (Real Only)**
- **الوصف:** يوليد صور حقيقية عبر الإنترنت فقط
- **السلوك:**
  1. يحاول Pollinations (AI generation)
  2. يحاول Pixabay (real photos search)
  3. يحاول Unsplash (real photos search)
  4. **يفشل إذا لم ينجح أي منهما** ❌
- **الفائدة:** جودة عالية جداً - صور حقيقية من مصادر موثوقة
- **متطلبات:** ✅ اتصال إنترنت ضروري
- **الاستخدام:** عندما تريد صور حقيقية عالية الجودة فقط

### 3. 📱 **محلي فقط (Local Only)**
- **الوصف:** توليد صور محلياً بدون الحاجة للإنترنت
- **السلوك:**
  - يستخدم محرك توليد الصور المحلي فقط
  - لا يحاول الوصول للإنترنت
  - ينتج صوراً سريعة وملونة
- **الفائدة:** 
  - ✅ يعمل بدون إنترنت
  - ✅ سريع جداً
  - ✅ لا يعتمد على API خارجية
- **الجودة:** صور محسّنة بـ Perlin Noise و Gradients (عالية الدقة 1024x1024)
- **الاستخدام:** 
  - بدون اتصال إنترنت
  - الأداء السريع المهم
  - الاختبار والتطوير

---

## 🎯 أنواع الصور المدعومة (محلياً):

تم تحسين التوليد المحلي ليدعم الأنواع التالية بصور **واقعية وعالية التفاصيل**:

- 👁️ **عيون** - عيون واقعية بتفاصيل دقيقة (بيضة العين، الحدقة، البؤبؤ، الأوعية الدموية، الإنعكاسات)
- 🐱 **قطط** - قطط ملونة مع عيون خضراء، أنف، آذان، وشعيرات
- 🐕 **كلاب** - كلاب بألوان طبيعية مع تفاصيل الرأس والجسم
- 🦅 **طيور** - طيور ملونة مع أجنحة ومنقار
- 🌄 **مناظر طبيعية** - سماء متدرجة، أرض، أشجار، وجبال
- 🌅 **غروب الشمس** - شمس ملونة مع انعكاسات على الأرض
- 👤 **بورتريهات** - وجوه بشرة طبيعية مع شعر وعيون
- 🌸 **زهور** - زهور ملونة مع ساق وأوراق

---

## 🚀 كيفية الاستخدام:

1. **اضغط على زر الصورة** 🎨
2. **أدخل وصفاً للصورة** (مثال: "قطة بعيون خضراء")
3. **اختر نوع التوليد** من القائمة المنسدلة:
   - 🔄 تلقائي (الافتراضي)
   - ☁️ حقيقي فقط
   - 📱 محلي فقط
4. **اضغط "توليد"** ✨

---

## 📊 مقارنة الخيارات:

| الميزة | تلقائي | حقيقي فقط | محلي فقط |
|------|--------|----------|---------|
| الجودة | عالية جداً | عالية جداً ⭐ | جيدة جداً |
| الحاجة للإنترنت | اختياري | ✅ ضروري | ❌ لا يحتاج |
| السرعة | متوسطة | متوسطة | سريعة جداً ⚡ |
| الضمان | ✅ سيعمل | متوقف على الإنترنت | ✅ سيعمل |
| الاستخدام الأمثل | الاستخدام العام | صور حقيقية عالية الجودة | بدون إنترنت |

---

## 💡 التوصيات:

- **للاستخدام العام:** استخدم **تلقائي** 🔄
- **للصور الاحترافية:** استخدم **حقيقي فقط** ☁️
- **بدون إنترنت:** استخدم **محلي فقط** 📱
- **للاختبار السريع:** استخدم **محلي فقط** 📱

---

## 🔧 التفاصيل التقنية:

### المتغيرات المستخدمة:
```dart
static const String MODE_AUTO = 'auto';        // تلقائي
static const String MODE_REAL_ONLY = 'real';   // حقيقي فقط
static const String MODE_LOCAL_ONLY = 'local'; // محلي فقط
```

### التطبيق:
```dart
// استدعاء الدالة مع اختيار الوضع
final result = await ImageService.generateImage(
  prompt,
  mode: ImageService.MODE_AUTO // أو MODE_REAL_ONLY / MODE_LOCAL_ONLY
);
```
